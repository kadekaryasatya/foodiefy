# Foodiefy
![foodiefy]()
## About Foodiefy
Sebuah platform penyedia layanan yang dibangun untuk mengelola resep masakan pengguna dan memberikan referensi masakan yang tersebar di seluruh dunia

### The Features: 
-Register 
-Login
-Explore Recipes
-My Recipes
-Add Recipes
-Delete Recipe
-Search Recipe
-Detail Recipe




### Project Resources : 
- React JS
- Bootstrap
- Spoonacular API
- Dicoding Notes API
